# 🧸 Fuzzy Factory – Business Performance Dashboard

📌 Project Summary

This project is an interactive Power BI dashboard created to analyze the business performance of Maven Fuzzy Factory, an e-commerce company selling toy products. The dashboard transforms raw website and order data into clear, visual insights that help understand how the business is performing across different areas.

It focuses on answering key business questions such as:

* Where is the traffic coming from?
* How well is the website converting visitors into customers?
* Which products generate the most revenue?
* How much revenue is affected by refunds?

 🎯 Purpose of the Dashboard

The main objective of this project was to practice real-world data analysis and build a dashboard that reflects how companies monitor their performance. It demonstrates how data can be used to track growth, identify trends, and support better decision-making.


🛠️ Tech Stack

* Power BI – for building the dashboard and creating visualizations
* SQL – for querying and preparing the data
* DAX – for calculated measures such as conversion rate, total revenue, and average order value


📂 Data Source

The dashboard is built using the **Maven Fuzzy Factory dataset**, which simulates real e-commerce operations. The data includes:

* Website sessions
* Customer orders
* Product information
* Refund records

This dataset allows the dashboard to replicate a realistic business intelligence scenario.

✨ Key Features

📊 Overview Page

* KPI cards showing total revenue, total orders, average order value, sessions, and conversion rate
* A trend chart that highlights how revenue and orders change over time

📈 Performance Page

* Traffic analysis showing how users reach the website (search, direct, social, etc.)
* Monthly order trends that reveal seasonal patterns in customer behavior
* Interactive buttons to switch between overview and detailed views

💰 Revenue Analysis Page

* Product-level revenue comparison
* Refund vs revenue analysis using percentage visuals
* Identification of top-performing and low-refund products

🔍 Funnel Insight

* A simplified funnel showing the journey from website sessions to completed orders, helping evaluate conversion efficiency.
